# F1 Visualization Dashboard

## Powered By

- [FastF1 Python Library](https://docs.fastf1.dev/index.html)

## Installation & Setup

### Activate environment

```bash
source f1-venv/bin/activate
```

### Install requirements

```bash
pip -r install requirements.txt
```

### Start the server

```bash
python app.py
```

## Deployment

### Heroku Deployment

```bash
git push heroku main
```
